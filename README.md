# FUTURE_DS_02
 Customer Retention & Churn Analysis

 Description
This dashboard analyzes customer churn behavior based on tenure, contract type, and services.

 Tools Used
- Power BI

 Files Included
- Power BI Dashboard (.pbix)
- Dataset
- Dashboard Screenshot

 Key Insights
- Churn Rate: 26.53%
- Highest churn in month-to-month contracts
- Customers with shorter tenure churn more
- Fiber optic users show higher churn
